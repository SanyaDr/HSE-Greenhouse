== Общее описание

Система предназначена для автоматического контроля микроклимата в теплицах на приусадебных участках.  
Она состоит из:

- **Конечных устройств**, устанавливаемых в теплицах (датчики + линейный привод + ESP32)
- **Облачной платформы** (ThingsBoard CE + кастомный бэкенд)
- **Веб-приложения** для управления и мониторинга

Основные функции:

- Измерение температуры и влажности
- Автоматическое открывание/закрывание форточки при превышении порогов
- Удалённое управление через веб-интерфейс
- Уведомления о критических событиях

== Поток данных

1. Устройство подключается к Wi-Fi и отправляет телеметрию в ThingsBoard по MQTT.
2. Пользователь заходит в веб-приложение → бэкенд запрашивает данные через REST API ThingsBoard.
3. При изменении настроек (например, нового порога температуры):
   - Веб → Бэкенд → ThingsBoard → MQTT-команда → Устройство.
4. В автоматическом режиме ThingsBoard может сам инициировать команду через Rule Chain.

== Безопасность

- Аутентификация устройств: **Access Token** (уникальный на устройство)
- Аутентификация пользователей: **JWT-токены** после входа
- Все соединения: **HTTPS / TLS**
- Данные пользователей изолированы (multi-tenancy через бэкенд)

== Масштабируемость

- Поддержка до 10 000 устройств на одном инстансе ThingsBoard CE (при достаточных ресурсах сервера)
- Горизонтальное масштабирование бэкенда
- Возможность перехода на ThingsBoard PE (Professional Edition) при коммерческом запуске
