=== Назначение
Автономный IoT-модуль для автоматического управления микроклиматом в теплице:

- Измерение температуры и влажности
- Управление линейным приводом (актуатором) для открывания/закрывания форточки
- Передача данных в облако и приём команд
- Работа в условиях повышенной влажности и перепадов температур

====  Программное обеспечение (прошивка)

- Поддержка Wi-Fi (WPA2)
- MQTT-клиент для подключения к ThingsBoard
- Отправка телеметрии: температура, влажность, положение привода, статус
- Приём команд: `open`, `close`, `stop`, `set_auto_mode`
- Локальная автоматика (если облако недоступно): открытие при T > 35°C
- OTA-обновления (Over-The-Air)

=== Электрическая схема (описание)

* **ESP32** управляет двумя GPIO:

  - `GPIO_OPEN` → активирует реле/драйвер «вперёд»
  - `GPIO_CLOSE` → активирует реле/драйвер «назад»
* При срабатывании **концевого выключателя** — подаётся сигнал на ESP32 → остановка привода.
  - Датчик **BME280** подключён по I²C (GPIO21 — SDA, GPIO22 — SCL).
  - Все цепи 12 В изолированы от 3.3 В.

=== Упаковка и поставка
Комплект поставляется в коробке с:

- Готовым к установке модулем в корпусе
- Крепёжными элементами (хомуты, шурупы)
- Инструкцией по монтажу и подключению
- QR-кодом для быстрой регистрации в веб-приложении
